Git Tips

  1. status: displays the status, useful to determine if already in git repo before initializing. [git status]
  2. init: creates new git repo in current directory. [git init]
  3. add: tells git to keep track of a file. [git add "file name"]
  4. config: used to configure git, mainly to list your name and email on commits. [git config --set user.name "your name"]
  5. commit: finalized changes. [git commit -m "commit message"]
  6. remote: used to manage files not directly in the git repo [git remote add "remote file"]
  7. push: sends git repo to a remote location, like the homework server [git push "server name"]
  8. log: prints a record of all git commits to the console [git log]

Bash Tips

  1. ls: displays contents of directory [ls -a]
  2. cd: changes directory [cd ~/Documents]
  3. rm: deletes a file [sudo rm -r *]

HTML Tips

  1. title: sets the name of the tab in the browser
  2. h1: first and largest header
  3. h2: second and second largest header
  4. p: paragraph
  5. code: use for displaying code
  6. a: anchor, used to create hyperlinks to other webpages
  7. img: used to insert images
  8. ol: used to start/end ordered lists
  9. ul: used to start/end unordered lists
  10. li: used to make an item in a list

CSS Tips

  1. background-color: makes the background not white, use hex-codes
  2. color: makes the text not black, use hex-codes
  3. font-family: specifies the font
  4. font-style: makes the font italic or oblique
  5. text align: changes text alignment
  6. margin: specifies how close an object can be to the edge of the page
  7. padding: specifies how close other objects can get
